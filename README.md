[control_error_mode](docs/control_error_mode.md) | Скрипт позволяет установить допустимый лимит ошибок для любых заездов. При «сходе с трассы» возможен автостарт следующего заезда | ![][i]
[NoErrorCounter](docs/NoErrorCounter.md) | Счётчик для безошибочного режима | ![][i]

[chat2BBCode](docs/chat2BBCode.md) | Кнопка, показывающая окно с логом текущего разговора в чате, возможность экспорта лога с форматированием BB-кодов | ![][i]
[KG_YandexTranslator](docs/KG_YandexTranslator.md) | Выводит перевод английских текстов в заездах при помощи сервиса «Яндекс.Перевод» |

[KG_AutoNextRace](docs/KG_AutoNextRace.md) | Добавляет возможность автоматического старта заезда и автосоздания следующего после набора |
